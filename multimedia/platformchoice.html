Platform Choice

Otago Polytechnic Bachelor of IT - Multimedia Development - Semester 2 2017

The reason I resorted to making pure html and css site thats hosted on github while forsaking jekyll was due to my lack of knowledge with the framework. Then I relised that opting for the "easy" path of onloading an HTML/CSS theme is the better option. However, this quickly changed when sifting through the hundreds of lines css code to do minute changes to the looks of the theme. However small, the learning struggle of navigating the endless CSS and HTML has solidified my knowledge of its inner workings. Also, I did not remain complacent with the theme I was provided. I made drastic changes to the page layout and formats to suit my style and personality. Also, there was no deying how responsive the drop down menu and pages are to mobile platforms. Thats besides the odd disproportional life hack video and 3d modal.
